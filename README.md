# Aexis Zero

A cross-platform interactive CLI for scaffolding modern apps with a terminal UI.

## Install (global)

```sh
npm install -g @aex.is/zero
```

## Install (Homebrew)

```sh
brew install aexis/homebrew-zero/zero
```

## Usage

```sh
zero
```

## Requirements

- Bun installed and available on PATH.

## Development

```sh
bun install
bun run dev
```
